# Rock Paper Scissors Game

## Project Overview

This is a simple and interactive **Rock Paper Scissors game** built using **HTML, CSS, and JavaScript**.

The game allows a user to play against the computer, where the computer randomly selects Rock, Paper, or Scissors. Scores are updated dynamically, and the game instantly shows whether the user wins, loses, or draws.

---

## Features

* Play Rock, Paper, Scissors against the computer
* Random computer choice generation
* Real-time score tracking
* Dynamic winner/loser messages
* Color-based result feedback
* Interactive and responsive UI

---

## Concepts Practiced

* DOM Manipulation
* Event Listeners
* Conditional Logic
* Random Number Generation
* Game State Management
* JavaScript Functions

---

## Technologies Used

* HTML
* CSS
* JavaScript

---

## Game Rules

* Rock beats Scissors
* Scissors beats Paper
* Paper beats Rock
* Same choices result in a draw

---

## How to Run

1. Clone or download the repository
2. Open the project folder
3. Run `index.html` in your browser
